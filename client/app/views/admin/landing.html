Welcome to GlobaLeaks Administration Interface

Clicking the 'admin' link on the top-right corner of the page will bring you here.

This is a work-in-progress of the administration interface that you will see after logging onto the platform as an admin.


This interface may help you understand how the Node can be configured and provides some info on how to connect to other GlobaLeaks users and experts.

  • You can reach GlobaLeaks's support chat via IRC on OFTC server on channel #globaleaks: GlobaLeaks Support Chat
  • To properly organize your initiative, please read the following guidelines: Guidelines on how to start up Whistleblowing Initiatives (English only)
  • To report a bug or request a feature in GlobaLeaks software, please open a ticket describing the issue you're encountering or the new feature request: GlobaLeaks Ticketing System
  • Remember to provide proper security advice to Whistleblowers: Security Advice for Whistleblowers (English only)
  • To know more about GlobaLeaks Security you can read the following document: GlobaLeaks Security (English only)
